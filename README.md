# Souvenir Tracker

This C++ program allows users to track their souvenir collections, calculate totals, and generate reports. The program includes user-defined functions, input validation, menu-driven selection, loops, and console colors for a more interactive experience.

## Features

- Friendly introduction banner
- Input validation for strings, integers, and doubles
- Calculates total cost per souvenir
- Menu with options:
  1. Add souvenirs
  2. View souvenir report
  3. Recommend collector level
- Generates a formatted report saved to `report.txt`
- Uses console colors to improve readability
- Demonstrates functions for input collection, validation, calculations, file saving, and displaying menus

## How to Use

1. Clone or download the repository.
2. Open the project in Visual Studio (or another IDE that supports C++).
3. Build and run the program.
4. Follow the on-screen menu to add souvenirs, view reports, or check your collector level.
5. A formatted report is automatically saved to `report.txt`.
